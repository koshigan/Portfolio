1. Line 5: `<title>`
2. Line 30: `.logo`
3. Line 190: `hero-title`
4. Anywhere else you mention it
5. Footer credit

### To Update Links
Edit `index.html`:
- GitHub: Search `github.com`
- LinkedIn: Search `linkedin.com`
- Email: Search `mailto:`

### To Update GitHub Username
Edit `script.js` line 1:
