## 🚀 **Quick Start**

### 1. No Installation Needed
Just open `index.html` in your browser!

### 2. Customize Your Info
Edit `index.html`:
- Change name "Koshigan" to your name
- Update email: `koshigan20@gmail.com`
- Modify GitHub link: `https://github.com/koshigan`
- Update LinkedIn: `https://www.linkedin.com/in/koshigan26`

### 3. Add Your Profile Picture
Place photo in `assets/profile.jpg` (400x400px recommended)

### 4. Update GitHub Username
In `script.js`, change:
```javascript
const GITHUB_USERNAME = 'your-github-username';
```

### 5. Deploy Online
- **Vercel** (Recommended) - `vercel deploy`
- **Netlify** - Drag & drop or connect GitHub
- **GitHub Pages** - Push to `gh-pages` branch
- **Railway/Render** - Modern alternatives

## 🎨 **Customization**

### Change Colors
Edit in `styles.css`:
```css
:root {
    --primary: #00d9ff;      /* Cyan */
    --secondary: #b300ff;    /* Purple */
    --accent: #ff0080;       /* Magenta */
}
```

### Modify Fonts
```css
--font-display: 'Sora', sans-serif;
--font-body: 'Inter', sans-serif;
--font-mono: 'Space Mono', monospace;
```

### Adjust Animations
```css
--transition-fast: 0.15s;
--transition-base: 0.3s;
--transition-slow: 0.6s;
```
