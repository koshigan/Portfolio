# Koshigan's Portfolio

A premium, modern portfolio website built with HTML, CSS, and Vanilla JavaScript.

## 🚀 Features
- **Voice Assistant Integration**: Showcasing a web-based voice assistant (Speech Recognition & TTS).
- **IoT & ECE Focus**: Highlighting a Cable Fault Prediction System using ESP32.
- **Modern UI**: Glassmorphism, Dark/Light mode theme toggle, and scroll-reveal animations.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Icons**: FontAwesome 6
- **Fonts**: Google Fonts (Outfit & Inter)

## 📂 Project Structure
- `index.html`: Main semantic structure.
- `styles.css`: Custom premium styling and animations.
- `script.js`: Interactive logic and theme handling.
- `assets/`: Contains the profile photo and other media.

## 🌐 Deployment
This portfolio is ready to be hosted on platforms like **Vercel**, **Netlify**, or **GitHub Pages**.
